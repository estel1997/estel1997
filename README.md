SwiftUI、MVVM、API 通信、Supabase、Flutter を使いながら、実装だけでなく「何を作ったか」「何を学んだか」「どう設計したか」を README と docs で説明できるポートフォリオ作りを進めています。

## Main Portfolio

- SwiftUI
- MVVM style
- URLSession

- JSONDecoder
- Supabase Ranking
- UserDefaults
- Error Handling
- Loading State

### Blue Archive Fan Quiz Web

HTML / CSS / JavaScript / Supabase / PostgreSQL / Cloudflare Pages を使った Web クイズアプリです。

- HTML / CSS / JavaScript
- Supabase
- PostgreSQL
- Cloudflare Pages
- Ranking
- Form Submission
- Public Deployment
- Security Notes

### OKITOKU

沖縄県内のスーパー・日用品価格を記録し、店舗ごとの価格比較を行うことを想定した Flutter アプリです。

- Flutter
- Dart
- Riverpod
- go_router
- Supabase
- Firebase / FCM verification
- Mobile App
- Price Comparison
